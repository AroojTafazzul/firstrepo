From this screen, the user is able to access the Transaction Bulk module provided by the Portal. The customer is able to create transaction bulks and handle transactions as a group and is able to create bulks from uploaded files as well.

Please note that there are cut-off timings for Bulk Services.

Note:

  • • Please avoid submitting files too close to the cut-off times to avoid missing the submission deadline and incurring delay.
Note:
  • • "V" denotes Value Date. This is the date when the payments are effected. Example: When salaries are to be made available for withdrawal by staff on 15 Jan 2002, the value date is 14 Jan 2002. On 14 Jan 2002, the staff accounts are credited by the receiving banks during their end-of-day processing (which is typically around midnight). For Collections, this value date is the date when the receiving accounts are debited.
  • • For Normal Service, value date cannot fall on Saturday, Sunday or Public Holiday, while for Express Service, value date cannot fall on Sunday and Public Holiday.
  • • Please avoid submitting files too close to the cut-off times to avoid missing the submission deadline and incurring delay.
