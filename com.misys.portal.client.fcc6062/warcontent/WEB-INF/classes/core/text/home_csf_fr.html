À partir de cet écran, un administrateur peut accéder à certaines Fonctionnalités du Système à des fins de maintenance. Suivant son profil et ses privilèges, l'utilisateur peut modifier diverses caractéristiques.

Les élements suivant peuvent être administrés :

  • Le Mot de Passe de l'utilisateur courant,
  • Les Utilisateurs et leurs privilèges respectives sont également maintenus par le biais de cet écran,
  • L'écran fournit également le moyen de maintenir les Données Statiques telles que les contre-parties clients ou banques, la liste de comptes et de textes prédéfinis (phrases), disponibles dans le système pour aider les phases de saisie de données lors du travail sur les dossiers de transactions.
