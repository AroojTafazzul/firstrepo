From this screen, the bank officer will be able to prepare Reporting Messages to the customer on its transactions initiated through Trade Portal, as well as create new files for specific products notifications.

The bank user is also offered with the capability of having access to a comprehensive set of Inquiries on the current transaction records. The bank officers may also need some static data to be defined in order to speed up their transaction data input, and the screen therefore provides a way to maintain these counterparties or phrases available in the system.
