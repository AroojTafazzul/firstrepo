À partir de cet écran, l'utilisateur Banque peut préparer les Messages Transactionnels destinés aux Clients et portant sur les transactions initiées au travers du Trade Portal. Les notifications de nouveaux dossiers sont également préparées par le biais de ces fonctions.

Cet utilisateur Banque a également la possibilité de consulter à travers le module de Requêtes les détails de tous les dossiers existants. Il est également possible pour l'utilisateur Banque de maintenir les jeux de Données Statiques lui permettant ainsi qu'à ses collègues d'améliorer les phases de saisie par une mise à disposition de données préparées.
