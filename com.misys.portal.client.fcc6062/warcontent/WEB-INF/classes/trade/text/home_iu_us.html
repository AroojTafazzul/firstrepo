The users can request for new Undertaking to be issued by the bank using the various options under Initiation menu like Application Form, Template and Copy from existing transaction.

The users can amend the Undertakings issued by the bank, send messages to bank regarding the Undertakings issued and amended.

The users can respond to the actions created by the Bank, edit requests before submission to approval, view the Undertakings pending for approval, view the Undertakings issued or rejected by the Bank.
