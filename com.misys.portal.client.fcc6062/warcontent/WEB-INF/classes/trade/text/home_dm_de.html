In diesem Bildschirm können Sie einen Ordner erstellen und die angehängten Export-Dokumente ausarbeiten : Warenrechnung, Ursprungszeugnis und so weiter. Sie können später diesen Ordner ändern und die Dokumente wiederausarbeiten.

Sie können jederzeit die Dokumente zu die Banke vorlegen.

Sie können auch ihre Ordner anfragen um zum Beispiel die angehängten Dokumente zu wiederausarbeiten.

Zuletzt können Sie ihre Ordner verwalten besonders um verfallenen Ordner zu löschen.

Sie können jederzeit die Hilfe erreichen dank einer festgeschalteten Ikone.
